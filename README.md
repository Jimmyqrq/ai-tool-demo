# AI Text Processing Tool

A simple Python-based tool for text summarization and rewriting, designed to demonstrate basic data processing and analysis logic.

## Overview
This project processes input text and generates summarized and rewritten outputs. It simulates basic natural language processing (NLP) tasks and demonstrates how raw text data can be transformed into structured insights.

## Features
- Summarize input text
- Rewrite text
- Command-line interaction for user input/output
- Simple text processing logic

## Technologies
- Python
- Basic string processing and data handling

## Example
Input:
"This is a long sentence that contains a lot of information..."

Output:
"Summary: ..."
"Rewritten: ..."

## Skills Demonstrated
- Data processing and transformation
- Logical problem solving
- Python programming fundamentals
- Basic text analysis workflow
